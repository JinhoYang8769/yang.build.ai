# yang.build.ai

## 소개

이 프로젝트는 AI 기술을 활용한 다양한 소프트웨어 개발 프로젝트를 설계·구현·기록하기 위한 개인 포트폴리오 저장소입니다.

생성형 AI, 데이터 분석, 자동화 도구 개발 등 실제 활용 가능한 서비스를 직접 구현하며 문제 해결 능력과 개발 역량을 향상시키는 것을 목표로 합니다.

## 주요 기능

- OpenAI API 기반 AI 서비스 개발
- 데이터 수집 및 분석 자동화
- Python 기반 백엔드 로직 구현
- Git/GitHub 기반 버전 관리
- 프로젝트별 기술 문서 작성

## 사용 방법

1. 저장소 다운로드

```bash
git clone https://github.com/[github-id]/yang.build.ai.git

2. 프로젝트 실행

cd yang.build.ai
python main.py

```markdown
